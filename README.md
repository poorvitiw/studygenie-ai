#  StudyGenie AI

An AI-powered study planner built using Spring Boot that helps students generate study plans, manage tasks, and track progress.

---

##  Features

*  Task Management (CRUD APIs)
*  Filter tasks by status & priority
*  AI-powered Study Plan Generator
*  Convert study plans into tasks
*  Mark tasks as completed
*  RESTful API design

---

##  Tech Stack

* Java 17+
* Spring Boot
* Spring Data JPA
* MySQL / H2
* REST APIs

---

##  API Endpoints

### Task APIs

* `POST /tasks`
* `GET /tasks`
* `PUT /tasks/{id}`
* `DELETE /tasks/{id}`
* `PUT /tasks/{id}/complete`

### AI APIs

* `POST /ai/explain`
* `POST /ai/summarize`
* `POST /ai/questions`
* `POST /ai/plan`
* `POST /ai/plan/save`
